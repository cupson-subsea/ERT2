## 🛰️ System Architecture Overview
LithoShield AI is a specialized geoscientific dashboard designed for real-time visualization of subsea and near-surface structural data. It simulates 2D Electrical Resistivity Tomography (ERT) inversion modeling and integrates interactive GIS frameworks to map subsurface geohazards and sedimentary basinal sequences.

### ⚡ Key Features
* **Interactive 2D Tomography Inversion:** Simulates real-time electrical resistivity profiling across distinct strata layers.
* **Dynamic GIS Mapping Module:** Leverages Leaflet.js to pinpoint target coordinate nodes dynamically across sedimentary environments.
* **Lithological Matrix Reference:** Embedded catalog featuring critical aggregate profiles (Basalt, Granite, Sandstone, Shale) and engineering application mappings.

### 🧪 Target Sedimentary Basins Supported
1. **Niger Delta Basin:** Characterized by alternating deltaic sand/shale sequences and shallow freshwater aquiferous layers.
2. **Gulf of Guinea Sector:** Configured for offshore subsea tracking, highlighting high-porosity marine silts and biogenic gas geohazards.
3. **Anambra Basin:** Tailored for localized sedimentary infill modeling, structural sandstones, and deeper coal seams.

### 🛠️ Tech Stack & Dependencies
* **Frontend:** Semantic HTML5, Custom Cyberpunk CSS variables.
* **Mapping Engine:** Leaflet JS via open-source CDN.
* **Visualization Layer:** HTML5 Canvas API running matrix data animations.
* **Deployment Pipe:** GitHub Pages automated build engines.
